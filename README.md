# benzo-console

The business console for **Benzo** — private payroll & treasury on Avalanche.
Organizations provision a managed treasury, run private payroll (eERC), manage
members, and (for auditors) decrypt the compliance ledger.

Frontend only (Vite + React + wagmi/viem). Backend, contracts, and infra live in
[`Miny-Labs/benzo`](https://github.com/Miny-Labs/benzo).

See [`AGENTS.md`](./AGENTS.md) for the stack, the backend it talks to, brand
tokens, and the contribution pipeline.

Entry for the Avalanche Team1 India Speedrun — Privacy on Avalanche.
